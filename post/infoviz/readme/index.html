What's going on in this graph?: 시작하며

Posted on 2017, Oct 12 One min read

뉴욕타임즈에서 학생들의 토론학습을 위해 제공하는 What’s Going On in This Graph? 번역 및 그래프 리뷰입니다. 보도했던 기사의 그래프 이미지를 본문은 생략한 채로 먼저 공개하고, 몇가지 질문에 대해 코멘트를 제출하면 이후 본문을 공개하는 방식의 컨텐츠입니다. 뉴스를 위한 데이터 시각화를 알아보고자 작성을 시작했으며 본문 번역과 제 의견, 가능하다면 해당 그래프 유형에 대한 정보까지 추가하려고 합니다.
